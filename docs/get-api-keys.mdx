Some hosts require an API key, while others support one optionally but don’t depend on it.
Here you can find out how to obtain your personal API key for each hosts requesting one.

:::warning
You must have an account with the host in question to obtain an API key.
:::

## 1fichier

- Go to: https://1fichier.com/console/params.pl
- Select "API Key - Assosciated Devices -"
- Click on the "Get my API key" button

![1fichier 1](/img/get-api-keys/1fichier_1.png)

## Drop.download

- Go to: https://drop.download/account/

The settings don't seem to work, so we need to access the configuration menu in a different way.

- Open the element inspector (F12 or Ctrl + Shift + I) and paste this into the search field: `.tab-content`

![Drop_download 1](/img/get-api-keys/drop_download_1.png)

- Cut the `show` and `active` classes from the first `my_account` element and paste them onto the `my_config` element

![Drop_download 2](/img/get-api-keys/drop_download_2.png)

![Drop_download 3](/img/get-api-keys/drop_download_3.png)

- Click on "Generate API key"

![Drop_download 4](/img/get-api-keys/drop_download_4.png)

(repeat the process to display the configuration menu)

- Your API key can be found in the URL: https://drop.download/api/account/info?key= `API key`

![Drop_download 5](/img/get-api-keys/drop_download_5.png)
