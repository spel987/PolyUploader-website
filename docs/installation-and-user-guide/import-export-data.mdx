You can **import** and **export** your history, API keys and profiles.

## Import data

Press the **“Import data” button** at the bottom of the page.

![Import data](/img/import-export-data/import_data.png)

Then click on the **purple "Import data" button**.

![Import data button](/img/import-export-data/import_data_button.png)

And select the **zip file** created during export by you or another PolyUploader user.

![Import zip](/img/import-export-data/import_zip.png)

## Export data

Press the **"Export data” button** at the bottom of the page.

![Export data](/img/import-export-data/export_data.png)

Then click on the **purple "Export data" button**.

![Export data button](/img/import-export-data/export_data_button.png)

And select the **name of the zip file** to be created.

![Export zip](/img/import-export-data/export_zip.png)
