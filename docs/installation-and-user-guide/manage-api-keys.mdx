You can import your **own API keys** for hosts requiring one. These hosts are at the end of the list and **cannot be used if you don't enter your API key**.

:::info[Info]

The screenshots shown are from an earlier version, so the interface may have changed slightly since then.

:::

![Set API keys buttons example](/img/manage-api-keys/set_api_keys_button.png)

## Import your API keys

Paste your **API key** and press the **red “Set API key” button**.

![Set API key user](/img/manage-api-keys/set_api_keys_user.png)

:::warning[warning]

Your API key **will not be tested for authenticity**. Please enter a valide API key to avoid upload errors.

:::

## Manage API keys

To manage your API keys, press the **purple “Manage API keys” button at the top of the page**. It is disabled if no API keys are imported by the user.

![Manage API keys button](/img/manage-api-keys/manage_api_keys_button.png)

You can then **view** and **delete** your API keys.

![Manage API keys](/img/manage-api-keys/manage_api_keys.png)
